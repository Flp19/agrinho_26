##  Descrição Geral do Projeto

O **AgroCode** é uma plataforma web desenvolvida com o objetivo de demonstrar como a tecnologia pode ser aplicada ao agronegócio para aumentar a produtividade, otimizar recursos e auxiliar produtores rurais na tomada de decisões. O projeto reúne conceitos de Agricultura 4.0, Internet das Coisas (IoT), Inteligência Artificial, monitoramento remoto e análise de dados em um ambiente digital intuitivo e moderno.

A proposta principal é permitir que agricultores tenham acesso a informações importantes sobre sua propriedade sem a necessidade de realizar análises manuais complexas. Através de uma interface amigável, o sistema oferece ferramentas para monitoramento de solo, irrigação inteligente, utilização de drones agrícolas, simulações de consumo hídrico e geração de relatórios técnicos.

O AgroCode foi pensado tanto para fins educacionais quanto para demonstrar aplicações reais que podem ser implementadas no setor agrícola, promovendo maior eficiência, sustentabilidade e redução de custos operacionais.

#  Objetivos do Projeto

O desenvolvimento do AgroCode possui diversos objetivos relacionados à modernização do campo e à utilização da tecnologia para melhorar processos agrícolas.

Entre os principais objetivos estão:

* Demonstrar a aplicação da programação na agricultura;
* Simular sistemas utilizados na Agricultura de Precisão;
* Auxiliar produtores na análise das condições do solo;
* Reduzir desperdícios de água por meio de irrigação inteligente;
* Apresentar o uso de drones para monitoramento agrícola;
* Aplicar conceitos de Inteligência Artificial em tomadas de decisão;
* Promover o aprendizado sobre tecnologias voltadas ao agronegócio;
* Facilitar o acesso de produtores às informações de suas propriedades;
* Incentivar práticas agrícolas mais sustentáveis.

#  Funcionalidades do Sistema

##  Sistema de Login

A primeira etapa do sistema é a autenticação do usuário. Essa área foi desenvolvida para representar um ambiente seguro onde o produtor pode acessar suas informações e ferramentas personalizadas.

O login permite que o usuário informe:

* Nome de usuário;
* E-mail de acesso;
* Senha protegida.

Além disso, o sistema conta com um botão de visualização da senha para facilitar o preenchimento dos dados e melhorar a experiência do usuário.

Futuramente, essa funcionalidade poderá ser integrada a bancos de dados online e sistemas de autenticação mais avançados.

---

##  Página Inicial (Home)

Após realizar o login, o usuário é direcionado para a página principal do sistema.

Nesta área encontram-se atalhos rápidos para todas as funcionalidades disponíveis na plataforma, permitindo uma navegação simples e intuitiva.

A tela inicial também apresenta:

* Nome do projeto;
* Identificação do usuário conectado;
* Nome do desenvolvedor;
* Cards de acesso rápido para cada módulo.

O objetivo desta seção é facilitar o acesso às ferramentas mais utilizadas pelos produtores.

---

##  Análise de Solo

A análise de solo é uma das funcionalidades mais importantes do AgroCode.

Nessa área, o sistema simula a coleta e interpretação de dados químicos do solo, exibindo indicadores fundamentais para a agricultura.

São analisados os seguintes nutrientes:

### Nitrogênio (N)

Responsável pelo crescimento vegetativo das plantas e pela formação de folhas saudáveis.

### Fósforo (P)

Importante para o desenvolvimento das raízes e para o processo de floração.

### Potássio (K)

Auxilia na resistência das plantas contra doenças e melhora a qualidade da produção.

O sistema também apresenta:

* Barras de progresso para visualização dos níveis dos nutrientes;
* Simulação de pH do solo;
* Diagnóstico automático baseado nos resultados obtidos;
* Recomendações para possíveis correções.

Essa funcionalidade demonstra como sensores e tecnologias modernas podem auxiliar agricultores na gestão eficiente do solo.

---

##  Monitoramento por Drones

O AgroCode também apresenta uma área dedicada ao uso de drones na agricultura.

Os drones vêm sendo cada vez mais utilizados para:

* Monitoramento de plantações;
* Identificação de pragas;
* Mapeamento de áreas agrícolas;
* Aplicação localizada de defensivos;
* Coleta de imagens aéreas.

Nesta seção, o sistema exibe informações simuladas de telemetria, como:

* Intensidade do sinal;
* Nível da bateria;
* Temperatura operacional;
* Área monitorada.

Além disso, há um link para uma palestra educativa sobre o uso de drones na agricultura moderna.

O objetivo é mostrar como os VANTs (Veículos Aéreos Não Tripulados) podem contribuir para uma agricultura mais eficiente e sustentável.

---

##  Irrigação Inteligente

A irrigação é um dos fatores mais importantes para garantir uma produção agrícola de qualidade.

Pensando nisso, o AgroCode possui um sistema de irrigação inteligente capaz de simular decisões automatizadas com base nos dados do solo.

Nesta seção o usuário pode:

* Monitorar o status da irrigação;
* Iniciar ciclos automáticos;
* Verificar o funcionamento dos sensores;
* Acompanhar o modo de operação do sistema.

O sistema utiliza uma interface inspirada em centrais de monitoramento agrícola modernas, proporcionando uma experiência mais realista.

O principal objetivo é demonstrar como sensores e automação podem reduzir o desperdício de água e aumentar a eficiência da produção agrícola.

---

##  Simulações Inteligentes

O módulo de simulações reúne três ferramentas importantes para auxiliar na tomada de decisões.

###  Monitor de Consumo

Analisa a umidade do solo informada pelo usuário e calcula a necessidade de irrigação.

Essa funcionalidade ajuda a evitar tanto o excesso quanto a falta de água.

---

###  Cálculo de Vazão

Permite calcular a quantidade estimada de água necessária para determinada condição do solo.

O resultado auxilia no planejamento do sistema de irrigação.

---

###  AgroCode AI

Esta funcionalidade representa a inteligência artificial do sistema.

O usuário informa:

* Nível de umidade do solo;
* Probabilidade de chuva.

A partir desses dados, a IA realiza uma análise e fornece recomendações sobre irrigação e manejo da área agrícola.

Embora seja uma simulação, essa ferramenta demonstra como algoritmos inteligentes podem auxiliar produtores na tomada de decisões mais precisas.

---

##  Relatório Final

Após utilizar as funcionalidades do sistema, o produtor pode acessar o relatório final.

Essa área reúne todas as informações geradas durante a utilização da plataforma.

O relatório apresenta:

* Dados da análise do solo;
* Informações do monitoramento por drones;
* Status da irrigação;
* Resultados fornecidos pela Inteligência Artificial.

Também existe a possibilidade de informar um endereço de e-mail para o envio do relatório técnico.

O objetivo é centralizar todas as informações em um único local, facilitando a consulta e o acompanhamento dos dados agrícolas.

##  Guia AgroTech

O Guia AgroTech funciona como uma biblioteca digital integrada ao sistema.

Nele o usuário encontra conteúdos educativos sobre temas relacionados à Agricultura 4.0.

Entre os assuntos abordados estão:

###  Irrigação Inteligente

Explica o funcionamento de sensores de umidade e sistemas automatizados de irrigação.

Demonstra como algoritmos podem prever produtividade e auxiliar na tomada de decisões.

O guia foi desenvolvido para servir como material de apoio para produtores, estudantes e interessados em tecnologia aplicada ao agronegócio.
